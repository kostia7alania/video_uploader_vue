### COMPOSER 
```
Если выполнить
composer install
то поставятся те версии которые указанны в composer.lock

.lock файл -  точно говорит какую версию поставить.

Поэтому рекомендуется писать
composer install
а не
composer update
т.к. update перезапишет .lock файл и добавит последнии версии.
```
